## How to Use

### GitHub Actions

```yml
name: Example
on: [push]
jobs:
  example:
    runs-on: ubuntu-latest
    container:
      image: docker://srzzumix/googletest:latest
    steps:
    - uses: actions/checkout@master
    - name: your build and test
      run: |
        make
        make test
```
